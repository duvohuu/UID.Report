\section{GIỚI THIỆU}

    \subsection{Tổng quan}

    \hspace*{0.6cm}Hệ thống \textbf{UID IoT Web Server} là nền tảng giám sát và quản lý máy móc công nghiệp theo thời gian thực, được phát triển bởi UID Lab với sự hợp tác của Ngọc Hiệp Leather Factory và SSB. Hệ thống cho phép người dùng theo dõi trạng thái, phân tích hiệu suất hoạt động của các thiết bị IoT từ xa thông qua giao diện web hiện đại.

    \hspace*{0.6cm}\subsection{Mục đích}

    Hệ thống được thiết kế để:
    \begin{itemize}
        \item Giám sát trạng thái máy móc theo thời gian thực
        \item Theo dõi các thông số vận hành (thời gian hoạt động, năng lượng tiêu thụ)
        \item Phân tích hiệu suất qua biểu đồ trực quan
        \item Quản lý người dùng và phân quyền truy cập
    \end{itemize}

    \subsection{Các loại máy được hỗ trợ}

    \begin{itemize}
        \item CNC Machine - Máy CNC
        \item Powder Filling Machine - Máy đóng gói bột
        \item Spray Machine - Máy phun sơn
        \item Salt Filling Machine - Máy đóng gói muối
    \end{itemize}

    \subsection{Phân quyền người dùng}

    \hspace*{0.6cm}Hệ thống hỗ trợ 2 cấp độ người dùng:

    \begin{itemize}
        \item \textbf{User}: Xem và theo dõi các máy được phân quyền
        \item \textbf{Admin}: Toàn quyền quản lý hệ thống, máy móc và người dùng
    \end{itemize}

    \subsection{Yêu cầu hệ thống}

    \begin{itemize}
        \item Trình duyệt web hiện đại (Chrome, Firefox, Safari, Edge)
        \item Thiết bị động có khả năng kết nối mạng (WiFi, Ethernet)
        \item Kết nối Internet ổn định
    \end{itemize}
